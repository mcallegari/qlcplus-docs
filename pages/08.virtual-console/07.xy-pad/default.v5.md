An **XY Pad** is a two-axis control for **pan and tilt** — drag the cursor around
the pad and the attached moving fixtures follow. It's the natural way to position
moving heads and scanners by hand from the [Virtual Console](VirtualConsole.md).

Add fixtures by dragging them onto the pad or from the settings; the pad's
horizontal axis drives **pan** and the vertical axis drives **tilt**.

## Settings

### Axis

For each of **Pan** and **Tilt**:

* **Minimum / Maximum** — limit the range of travel the pad uses, so you can
  confine movement to the stage area.
* **Reverse** — invert that axis's direction.

### Display Properties

* **Inverted Y-Axis** — flip the vertical axis so dragging up tilts down (or vice
  versa), to match how you think about the rig.
* **Range Display Mode** — how positions are shown: **Degrees**, **Percentage**
  or **DMX**.

### Fixtures

The fixtures (or heads) controlled by the pad:

* **Add a fixture/head** — add fixtures to the pad.
* **Set the Pan/Tilt range of the selected fixture head(s)** — define each
  fixture's usable **X-Axis Range** and **Y-Axis Range**, so fixtures with
  different ranges line up on the pad.
* **Remove the selected fixture head(s)** — remove fixtures from the pad.

## Presets

The XY Pad supports **presets** (a tab in its properties) — saved positions and
effects you can recall:

* Create a **position preset** from the current XY position.
* Drop **Scene / EFX** functions onto the pad to use them as presets.
* Drop **fixture groups / heads** to create fixture-group presets.

## Tips

* Set per-fixture **X/Y ranges** so a mixed rig of moving heads all point at the
  same spot when the pad cursor is centred.
* Save common positions as presets for instant recall during a show.

A **Clock** widget shows time on the [Virtual Console](VirtualConsole.md) — as a
running clock, a stopwatch or a countdown — and can **schedule functions** to
start and stop at set times. It's used for timed and unattended shows (e.g. a
museum or shop window that runs to a daily schedule).

## Settings

### Clock type

* **Clock** — shows the current time of day.
* **Stopwatch** — counts up from zero; useful for timing a running show.
* **Countdown** — counts down to zero from a set time.
* **Schedule** — runs functions according to a list of scheduled times (below).

### Schedule *(Schedule type)*

Build a list of timed events. For each schedule entry:

* **Add a function schedule** — attach a function to a new schedule entry.
* **Start time** — when the function starts.
* **Stop time** — when it stops (enable it with **Enable the stop time**; without
  it the function simply starts at the start time).
* **Days / Repeat weekly** — choose the days of the week the schedule runs, so an
  event can repeat on, say, every weekday.
* **Remove this schedule** — delete an entry.

Use **Add a new schedule** to add further entries.

## Tips

* Use **Schedule** with **Repeat weekly** to run a venue's lighting automatically
  on a recurring timetable.
* Use **Countdown** on stage as a visible timer for set changes or interval ends.

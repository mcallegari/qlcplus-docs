A **Speed Dial** sets and adjusts the **timing** of functions live — the fade in,
fade out and duration times. Use it to speed up or slow down chases and effects
on the fly, often tapped in time with the music. It belongs to the
[Virtual Console](VirtualConsole.md).

## Settings

### Functions

The list of functions whose timing this dial controls. **Drag and drop**
functions onto the widget to add them. For each function you choose which of its
times the dial drives:

* **Fade In Factor**
* **Fade Out Factor**
* **Duration Factor (+tap)** — the duration, which can also be set by tapping.

Each factor can be **(Not sent)** or a multiplier: **0, 1/16, 1/8, 1/4, 1/2, 1,
2, 4, 8, 16** — so one dial can, say, run one function at half speed and another
at double. Use **Remove this function** to drop a function from the list.

### Dial Properties

* **Reset multiplier factor when the dial value changes** — resets the applied
  multiplier whenever you set a new time on the dial.
* **Dial time range** — the **from / to** limits of the time the dial can set.

### Appearance

Choose which controls the widget shows:

* **+/- Buttons** — increment/decrement buttons.
* **Dial** — the rotary dial.
* **Tap** — a tap button to set the time by tapping a tempo.
* **Multipliers** — buttons to apply ×/÷ multipliers.
* **Apply** — an apply button.

### Multipliers / time units

The dial can work in **Hours**, **Minutes**, **Seconds**, **Milliseconds** or
**Beats**, so it can be a precise time setter or a beat-synced tempo control.

## Tips

* Set the time unit to **Beats** and use **Tap** to lock chases to the music's
  tempo.
* Assign several functions with different multiplier factors to ride a whole
  group of effects from one dial.

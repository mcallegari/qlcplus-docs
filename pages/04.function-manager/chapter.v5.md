The **Function Manager** is the list of all the **functions** in your project —
Scenes, Chasers, Sequences, EFX, Collections, RGB Matrices, Shows, Scripts,
Audio and Video. It lives in the right panel of the
[Fixtures and Functions](FixturesAndFunctions.md) workspace; open it with the
**Function Manager** button.

From here you select functions to edit, organise them into folders, and filter
or search the list. The buttons that create, delete, rename, clone and preview
functions are on the right panel itself — see
[Fixtures and Functions](FixturesAndFunctions.md#right-panel--functions).

## The list

Functions are shown in a tree. You can group them into **folders** to keep large
projects tidy.

* **Click** a function to select it; **Ctrl+click** to select several.
* **Double-click** a function to open its editor. (Double-clicking a **Show**
  switches to the Show Manager.)
* **Double-click** a folder to expand or collapse it.
* **Drag** functions onto a folder to move them into it; drag a function out to
  reorganise. You can also rename a folder to move everything it contains.
* **Click an empty area** below the list to clear the selection.

## Filtering by type

The toolbar across the top has one toggle button for each function type, each
showing a **count** of how many functions of that type you have:

* **Scenes**
* **Chasers**
* **Sequences**
* **EFX**
* **Collections**
* **RGB Matrices**
* **Shows**
* **Scripts**
* **Audio**
* **Videos**

Toggle a type off to hide those functions from the list, making it easier to
focus on what you're working on.

## Search

The **search** button (magnifier) at the right of the toolbar opens a box where
you can filter the list by function name.

## Editing a function

Selecting and then editing (or double-clicking) a function opens its dedicated
editor in the right panel. Each function type has its own editor — for example
the Scene editor, the Chaser editor, the EFX editor, and so on.

A **Video** function plays a video file (or a still picture, or a network
stream) on a screen. It lets QLC+ drive projection and screen content together
with the lighting. The Video Editor opens in the right panel of the
[Fixtures and Functions](FixturesAndFunctions.md) workspace; when you create a
video function you are asked to pick the file first.

## Properties

* **Name** — the function's name (top bar).
* **File name** — the video or picture file to play. Use the **...** button to
  choose a file, or the **globe** button to enter a **URL** for a network
  stream.
* **Duration**, **Resolution**, **Video Codec**, **Audio Codec** — read-only
  details of the selected media.
* **Playback mode** — **Single shot** (play once) or **Looped** (repeat).
* **Output screen** — which connected screen/monitor the video is shown on.
* **Output mode** — **Windowed** or **Fullscreen**.
* **Geometry** — **Original** (use the media's own size) or **Custom**. With
  Custom selected you can set:
  * **Position** — the **X** / **Y** position of the video window.
  * **Size** — the **W** / **H** width and height of the video window.
* **Rotation** — rotate the video around the **X**, **Y** and **Z** axes
  (degrees).
* **Layer** — the stacking order (1–100) when several videos play on the same
  screen; higher layers appear on top.

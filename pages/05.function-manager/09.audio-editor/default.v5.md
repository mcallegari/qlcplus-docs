An **Audio** function plays a sound file. It lets you trigger music, stings and
effects from QLC+ alongside your lighting. The Audio Editor opens in the right
panel of the [Fixtures and Functions](/fixtures-and-functions) workspace; when
you create an audio function you are asked to pick the file first.

## Properties

* **Name** — the function's name (top bar).
* **File name** — the audio file to play. Use the **...** button to choose a
  different file.
* **Duration**, **Channels**, **Sample Rate**, **Bitrate** — read-only details
  of the selected file, shown for reference.
* **Playback mode** — **Single shot** (play once) or **Looped** (repeat until
  stopped).
* **Output device** — which audio output the sound is sent to.
* **Volume** — playback volume, 0–100%.
* **Fade in** — time to fade the sound up when it starts. Double-click the field
  or use the clock button to set it.
* **Fade out** — time to fade the sound out when it stops.
